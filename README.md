# TRABAJO-FINAL-
sistema de monitoreo ambiental y seguridad de una sala neonatal 
Para ejecutar el código, primero se carga el programa en el ESP32 utilizando Arduino IDE o PlatformIO. En el entorno de desarrollo se selecciona la placa ESP32 Dev Module y el puerto correspondiente. Luego, se instalan las librerías necesarias (sensores, WiFi, Servo, etc.) y se compila el proyecto para verificar que no existan errores. Finalmente, se conecta el ESP32 por USB y se presiona Upload para transferir el código al microcontrolador.

Una vez cargado, el ESP32 inicia automáticamente la ejecución del programa. El sistema comienza leyendo los sensores, activando los actuadores y estableciendo la conexión WiFi. Desde el monitor serial se pueden visualizar las lecturas en tiempo real y confirmar que el dispositivo está enviando datos y generando alertas según las condiciones programadas.
